Server Set Up

Dịch vụ setup, thiết kế hệ thống quản lý, tùy chỉnh giao diện server Discord với nhiều gói dịch vụ phù hợp nhiều nhu cầu khác nhau, dành cho các hội nhóm cá nhân muốn xây dựng cộng đồng nhỏ đến doanh nghiệp, creator,…

Design

Thiết kế avatar server, banner server riêng hoặc thay đổi bộ nhận diện theo chủ đề/theme sự kiện, poster/banner sự kiện, emoji, hình ảnh minh họa,… dạng ảnh tĩnh hoặc ảnh động theo yêu cầu.
